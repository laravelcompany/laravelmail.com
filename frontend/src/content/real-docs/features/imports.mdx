## CSV Import

You can import a CSV file into Laravel Mail using the `Import Subscribers` button.

![https://i.imgur.com/6ePMQ2Y.png](https://i.imgur.com/6ePMQ2Y.png)

Once you click the Import button, you will be prompted to select the CSV file.

The csv file should have the following columns:

- `email` - required
- `first_name` - optional
- `last_name` - optional

![https://i.imgur.com/l4wXjAF.png](https://i.imgur.com/l4wXjAF.png)
